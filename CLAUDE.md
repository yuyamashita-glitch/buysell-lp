# バイセル 出張買取 折込チラシ・LP制作プロジェクト

## プロジェクト概要
バイセルの**出張買取**に特化した、商材別の折込チラシ（B4サイズ想定、表裏2面）およびそれを受けるLP（ランディングページ）をHTML/CSSで制作するプロジェクト。
様々な商材（おもちゃ・レゴ・ゲーム機・着物・貴金属・ブランド品・古銭・切手・お酒・骨董品・楽器など）の出張買取チラシとLPを展開していく。

※ 買取催事の折込チラシは別プロジェクトで管理。ここでは扱わない。

## 技術仕様
- 単一HTMLファイル（CSS埋め込み）で1チラシ
- フォント: Noto Sans JP (Google Fonts)
- 印刷用途: B4サイズ（257mm × 364mm）を想定したレイアウト
- デプロイ: GitHub Pages (yuyamashita-glitch/buysell-lp)
- 画像: `img/` ディレクトリに配置

## ファイル命名規則
- `chirashi_{商材名}.html` （例: chirashi_lego.html, chirashi_toy.html）
- 商材バリエーション: `chirashi_{商材名}_only.html`

## チラシ共通構成
- **表面**:
  - ヘッダー: 公式ブランディング画像(`img/header_official.jpg`)を使用。自作しない
    - CSS: `background: #f5d000; margin: 0; padding: 0; line-height: 0; overflow: hidden;`
    - img: `width: 100%; display: block; margin-bottom: -4px;`（白隙間防止）
    - 画像更新時は `?v=数字` のキャッシュバスターを付与
  - キャッチコピー → メイン商材画像 → 買取対象品目 → CTAボタン（電話番号）
  - 「現金即払い※」には※マークを付ける
- **裏面**:
  - 買取実績 → サービス特徴 → その他買取品目
  - 注釈セクション（`.disclaimers`）: 振込対応の注記、未成年買取不可、本人確認書類、季節・地域制約、宅配・店舗案内
  - 運営会社情報（`.company-info`）: 3行に分けて記載
    - 運営会社：株式会社BuySell Technologies
    - 〒160-0004 東京都新宿区四谷4-28-8 PALTビル
    - 【東京都公安委員会】古物商許可番号：第301041408603号
  - フッター（電話番号・査定士画像）
- **共通要素**: 赤(#c41e24)・緑(#1a6b2a)の配色、キャッシュ無効化メタタグ

## デプロイ手順
1. `cd ~/projects/buysell-lp && git add -A && git commit -m "メッセージ" && git push`
2. GitHub Pages ビルド確認: `gh api repos/yuyamashita-glitch/buysell-lp/pages/builds/latest --jq '.status'`
3. URL共有時は `?v=数字` のキャッシュバスターを付与

## コンテンツレギュレーション（公式ルールブック）

バイセル公式のコンテンツレギュレーションに必ず準拠すること。
詳細: `memory/buysell_content_regulation.md` に全文記録済み。

### 最重要NG表現（チラシ制作時に特に注意）
- **「高額」NG** →「高価」は条件付きOK
- **「どんな〇〇でも買い取ります」NG** →「どんな〇〇でも査定します」はOK
- **「引取」「処分」NG** → バイセルは無償引取・処分をしていない
- **「断捨離」NG** → 商標。代替：大掃除／整理術／捨て活
- **「鑑定士」NG** →「査定士」が正しい
- **「即日査定」NG** →「最短 即日査定」（「最短」必須）
- **「24時間365日 買取中」NG** →「24時間365日 受付中」
- **他社比較の断定NG** →「他社に断られても諦めないで！」はOK
- **断定表現（一切・全て・絶対）NG**
- **「不用品」NG**（「不要品」は制限付きOK）
- **オファー煽りNG**: 今だけ/期間限定/今月まで/お急ぎください/ラストチャンス 等

### 必須注釈チェックリスト
- [ ] 現金即払い → ※振込対応の場合あり
- [ ] 全国対応 → ※季節・地域により対応難しい場合あり
- [ ] 買取実績 → ※一例です。状態により金額変動
- [ ] 未成年買取不可
- [ ] 本人確認書類（運転免許証・マイナンバーカード・健康保険証）
- [ ] 宅配・店舗案内の注記
- [ ] 古物商許可番号（会社名とセットで表記）

### 表記ゆれルール
- 折込（おりこみ）、買取（名詞）、お買い取り（動詞）
- お客様、お片付け、お住まい、当せん、ください、いただく

## 改善ログ（自律改善の仕組み）

ユーザーから修正依頼を受けた場合、以下のプロセスを実行する：

### 修正時の行動規範
1. **修正依頼を受けたら**、まず `LESSONS.md` に記録する（何を間違えたか・なぜ・正しい対応）
2. **同じ系統のミスを繰り返さない**ため、新規チラシ作成時は必ず `LESSONS.md` を参照する
3. **修正パターンが3回以上蓄積した項目**は、この CLAUDE.md の「制作ルール」セクションに昇格させる

### 制作ルール（修正経験から学んだルール）
<!-- LESSONS.md から昇格したルールをここに追記していく -->
- 画像は `object-fit: contain` を基本とする（見切れ防止）
- 買取金額の表示は慎重に（実際と乖離しないよう確認を取る）
- 裏面下部の「その他買取品目」は全パターン共通で配置する
- 商標ルール: 「レゴ」単独NG →「レゴ®ブロック」。商標は前後と同じ書体で、強調表示禁止。LEGOロゴ使用禁止。他メーカーも商標ルール要確認
- ヘッダーは公式画像を使う。自作のHTML/CSSヘッダーは使わない
- 画像素材はバイセル保有の商品素材 or 自前撮影素材を使用。外部サイトの画像URLは著作権リスクあり
- 「現金即払い」には必ず※を付け、裏面に「※お支払いは内容によって振込対応となる場合もございます。」を記載
- 裏面に運営会社・古物商許可番号（第301041408603号）・各種注釈を必ず記載
- 画像ファイル更新時はHTMLのsrc属性にもキャッシュバスター(`?v=数字`)を付与する
